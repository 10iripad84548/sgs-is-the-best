    • Settings
    This is the settings page, change settings for the site here!

    Personalization
    Personalize the site to your likings.
    Mode
    Change the website's appearance by a dark and a light contrast.

    Theme Color
    This changes the website's main color that you might see when using light or dark themed mode.

    Tab Cloak
    This changes the website icon and title to help hide this among your tabs. Note: This may not be working correctly.

    Show Game Thumbnail
    This toggles whether to show the game thumbnail or not. This can also improve faster loading times and help reduce network usage.

    Hide Alt Links Button
    If you prefer it, this will hide the alt links button on the sidebar.


    Other
    Other general site settings. Some of this might be useful others not.
    Open Sidebar by Default
    This will keep the sidebar open and only close when you play a game or if you close it.
